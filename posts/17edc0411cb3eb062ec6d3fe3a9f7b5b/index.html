Linux系统设置vnc开机自启

1、安装tigervnc服务器

yum install -y tigervnc-server

2、启动vncserver

vncserver

命令输出如下：

You will require a password to access your desktops.
 
Password:
Verify:
Would you like to enter a view-only password (y/n)? n
A view-only password is not used
 
New 'hdp1:1 (root)' desktop is hdp1:1
 
Creating default startup script /root/.vnc/xstartup
Creating default config /root/.vnc/config
Starting applications specified in /root/.vnc/xstartup
Log file is /root/.vnc/hdp1:1.log

首次启动vncserver时提示输入设置远程连接密码，可以不同于本地密码，该密码以后可以使用vncpasswd命令重置。注意当出现 Would you like to enter a view-only password (y/n)? 提示时输入n。

# 查看vncserver进程列表
vncserver -list

 命令输出如下，显示当前启动了一个VNC服务器进程：

TigerVNC server sessions:
 
X DISPLAY #    PROCESS ID
:1        2431

3、设置自动启动的vncserver服务

vi /etc/rc.local

su - root -c "/usr/bin/vncserver"
