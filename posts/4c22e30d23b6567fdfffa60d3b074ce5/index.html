后端人员常用linux命令(持续更新)

后端人员常用linux命令(持续更新)
1.cd 命令
    说明: 切换当前目录
    参数：要切换到的目录路径
    例如: cd /usr/bin   # 切换到目录/usr/bin
          cd ./path        # 切换到当前目录下的path目录中， “.”表示当前目录
          cd ../path    # 切换到上一级目录下的path目录中， “..”表示当前目录
              cd ~        # 切换到家目录，或者直接cd
2.ls 命令
    说明：全拼为list, 查看目录和文件的命令
    参数：-l:列出长数据串，包含文件的属性与权限数据等
          -h:列出长数据串，以人们可以看懂的文件大小展示(M,K,G),常与-l组合使用        
          -a:列出全部的文件，连同隐藏文件（开头为.的文件）一起列出来（常用）
    例如：ls -l
    结果:drwxr-xr-x 2 happy happy      4096 3月  20 10:03  C
        第一个字段：文件属性字段，共10个字母组成
            第一个字符代表文件的类型('-':表示普通文件,'d':表示目录文件)
            后9个字母表示该文件或者目录的权限位
            r表示读(Read),w表示写(Write),x表示执行(eXecute)
            其中前三个表示文件所有者权限，中间三个表示文件所属组拥有的权限，最后三个表示其他用户拥有的权限
            示例表示：文件所有者有读写执行权限，文件所属组有读和执行权限，其他用户有读和执行权限
        第三个字段：文件（目录）拥有者
        第四个字段：文件（目录）拥有者所在组
        第五个字段：文件所占用的空间(以字节为单位)
        第六个字段：文件（目录）最近访问（修改）时间(3月 20 10:03)
        第七个字段：文件名
3.pwd 命令
    说明：全拼print working directory,确定当前目录路径
    参数：无
4.cp 命令
    说明：全拼copy，复制文件
    参数：-a:连同文件的特性一起复制
          -p:同-a常用于备份
              -i:若目标文件已经存在时，在覆盖时会先询问操作的进行
          -r:递归持续复制，用于目录的复制行为
          -u:目标文件与源文件有差异时才会复制
    例如：cp file1 file2       # 将文件file1复制成文件file2
          cp file1 file2 dir   # 将文件file1 ,file2复制到文件夹dir
5.mv 命令
    说明：全拼move,用来移动文件到指定目录或者更改文件名字
    参数： -f ：force强制的意思，如果目标文件已经存在，不会询问而直接覆盖
               -i ：若目标文件已经存在，就会询问是否覆盖
               -u ：若目标文件已经存在，且比目标文件新，才会更新
    例如：mv file1 file2 dir    # 将文件file1,file2移动到目录dir中
          mv file1 file2        # 将文件file1重命名为file2
6.rm 命令
    说明：用于删除目录或者文件
    参数：  -f ：就是force的意思，忽略不存在的文件，不会出现警告消息
            -i ：在删除前会询问用户是否操作
            -r ：递归删除，最常用于目录删除，它是一个非常危险的参数
    例如： rm -rf dir    # 强制递归删除dir目录， -r -f 常组合使用
7.touch 命令
    说明：新建文件
    例如：touch filename  # 新建名为filename的文件
8.mkdir 命令
    说明：新建文件价或者目录
    例如：mkdir dirname  # 新建名为dirname的文件夹
9.scp 命令
    说明：全拼secure copy, scp 是 linux 系统下基于 ssh 登陆进行安全的远程文件拷贝命令。
    参数：-r： 递归复制整个目录
          -P port：注意是大写的P, port是指定数据传输用到的端口号
    例如：scp local_file remote_username@remote_ip:remote_folder # 将本地文件复制到远程服务器
          scp remote_username@remote_ip:remote_folder file       # 将远程服务器文件复制到本地
10.export 命令
    说明：修改环境变量
    例如：export PATH=$PATH:\libpath     # 将libpath添加到PATH
