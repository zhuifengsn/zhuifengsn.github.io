2016 级《计算机导论》考试试题A卷

考试时间：2017年1月5日班级教学号姓名

?要求答案写在答题纸上，写上题号，不必抄题，英文答卷；

?要求在答题纸和试题纸上都写上班级、学号和姓名，交卷时一并交上来；

?要求在每个大题号(I, II, III,IV,V)前划一个满页的横线，以便教师批阅。

I、Choice Questions(20 questions, 1 score for each question)

1. The step by step solution that expresses the logical solution to a particular problem of interest is called a(n) ________.

A. algorithm

B. answer

C. program

D. process

2. The ASCII character set includes ________ characters.

A.256

B. 128

C. 65536

D. 4,294967296

3. The most common way to represent signed integers in computer science is ________.

A. sign-and magnitude

B. one’s complement

C. two’s complement

D. unsigned integers

4.What is -31 in Excess_127 notation? ________

A. 01100001

B. 11100001

C. 10011111

D. 01100000

5. Computer is based on the ________ model..

A. Intel

B. Microsoft

C. input/output

D. von Neumann

6. The place where programs and data are stored during processing is ________..

A. display

B. main memory

C. floppy disk

D. hard disk

7. A computer stores data and program as a ________ pattern.

A. unary

B. hexadecimal

C. binary

D. numeric

8. A byte consists of ______ bits.

A. 2

B. 4

C. 8

D. 16

9. If the memory address space is 32MB and the word size is 8 bits, then ______bits address are needed to access each word.

A. 25

B. 16

C. 8

D. 22

10. The three steps in the running of a program on a computer are performed ______in this specific order.

A. fetch, decode, and execute

B. fetch, execute, and decode

C. decode, fetch, and execute

D. decode, execute, and fetch

11.A ______is a device that can route a packet based on its network layer address.

A. bridge

B. router

C. repeater

D. all of the above
