当vue里面props属性type为数组或者对象需要设置默认值为空数组或对象

一、vue官方文档解释：为该 prop 指定一个默认值。如果该 prop 没有被传入，则换做用这个值。对象或数组的默认值必须从一个工厂函数返回。
1.默认值为空数组

props:{
	ary:{
		type:[Array],
		required:true,
		default: () => []
	}
}

2.默认值为空对象

//正确写法
props:{
	obj:{
		type:[Object],
		required:true,
		default: () => ({})
	}
}

//错误写法报错   需要用小括号包起来
props:{
	obj:{
		type:[Object],
		required:true,
		default: () => {}
	}
}
