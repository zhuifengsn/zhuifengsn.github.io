修改时间

Linux

ntp自动校时

  • 关闭ntp自动校时：sudo timedatectl set-ntp no
  • 打开ntp自动校时：sudo timedatectl set-ntp yes

date -s

  • 修改系统时间
sudo date -s "2016/11/11 10:21:32"
sudo date -s "2016-11-11 10:21:32"
sudo date -s "20161111 10:21:32"
  • 另外也可以使用 next 或 last 的方式來設定
sudo date -s "next Mon" # 下週一
sudo date -s "last Mon 17:30:21" # 上週一 17:30:21

timedatectl

  • 修改时间：sudo timedatectl set-time "2016-11-12 18:10:40"

MacOS

  • 关闭自动设置时间

  • 修改的时间，都是使用两位数为替代，分别是 月、日、小时、分钟和年份。
    sudo date 1019121019
