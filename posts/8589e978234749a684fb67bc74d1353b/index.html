[安装fcitx和搜狗输入法后]Ubuntu18.04开机全黑屏，只有鼠标可移动

遇到的问题

某次重启进入Ubuntu，全黑屏，只有鼠标能移动；我以为是系统坏了，手动重启进入还是这个现象

 

问题缘由

安装了中文输入法，采取的是fcitx和搜狗输入法；

我不清楚这俩输入法为啥在这个ubuntu系统里会出现这种情况（之前我在虚拟机和Tx2的ubuntu系统里都没有遇到过）

 

解决办法

首先ctrl+alt+F6进入命令行的界面（如若F6不行就走F2）

sudo apt-get remove sogoupinyin
sudo apt-get autoremove 
reboot

如果卸载搜狗之后重启，还是这个问题，那么接着删

sudo apt-get remove fcitx
sudo apt-get autoremove //自动移除无用的包。
reboot

然后，我能够解决这个问题；如果还不行，请往下看
