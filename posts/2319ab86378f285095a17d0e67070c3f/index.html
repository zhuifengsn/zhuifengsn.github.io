1、关于nth-of-type、nth-child选择失败的问题
如下示例：会发现.b:nth-of-type(1) { color: red; }失效

<div class="demo">
	<div class="a">aa</div>
	<div class="b">b1</div>
	<div class="b">b2</div>
	<div class="c">cc</div>
</div>

.class:nth-of-type(i) 的真正意义是：
要选中的元素的类名是class
这个元素是它的父元素下同一个标签名的元素中的第 i 个

2、every方法判断空数组失效

[].every(i => i === 1)  // 打印出来是true，并不是false
[2, 1].every(i => i === 1)  // 打印出来是false
[1, 1].every(i => i === 1)  // 打印出来是true
