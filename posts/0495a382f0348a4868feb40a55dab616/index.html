蓝桥杯-四平方和（python）

1.题目

 

2.python实现

import math
n=int(input())
xi=int(math.pow(n/4,1/2))
xj=int(math.pow(n/3,1/2))
xk=int(math.pow(n/2,1/2))
def f(n):
  for i in range(xi):
    for j in range(xj):
      for k in range(xk):
        t=n-i*i-j*j-k*k
        m=int(t**0.5)
        if m>=k and i*i+j*j+k*k+m*m==n:
            print(i,j,k,m,sep=' ')
            return
f(n)

 3.代码解释

  1. 导入了 math 库，以便使用其中的 pow 和 sqrt 函数。

  2. 通过 int(input()) 从控制台读取一个整数，将其赋值给变量 n。

  3. 通过 pow 函数和 sqrt 函数计算出 n/4、n/3 和 n/2 的平方根，将其分别赋值给变量 xi、xj 和 xk。这些平方根将用于限制循环中 i、j 和 k 的取值范围，从而加速计算。

  4. 定义一个名为 f 的函数，该函数的参数为整数 n。

  5. 在函数 f 中使用三重嵌套的 for 循环，分别枚举 i、j 和 k 的取值范围。其中，i、j 和 k 的取值范围分别是 [0, xi)、[0, xj) 和 [0, xk)。

  6. 在内层循环中，计算 t=n-ii-jj-k*k，并将其赋值给变量 t。

  7. 使用 sqrt 函数计算出 t 的平方根，将其转换为整数类型，并将其赋值给变量 m。

  8. 如果 m 大于或等于 k，且 ii+jj+kk+mm 等于 n，则输出 i、j、k 和 m，并使用 sep 参数将它们之间的空格替换为一个空格。

  9. 在输出结果后，使用 return 语句退出函数。

  10. 最后，在主程序中调用函数 f，将变量 n 的值传递给它，从而求解出四个数平方和等于 n 的问题。
