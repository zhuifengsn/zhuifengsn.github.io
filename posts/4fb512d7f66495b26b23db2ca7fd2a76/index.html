2，将stata11转换为stata15需要解决中文乱码问题

命令为：
. cd “D:\stata15\ado\personal\mypaper” // 请自行修改路径
. ua: unicode encoding set gb18030
. ua: unicode translate *

详见以下链接：
https://blog.csdn.net/arlionn/article/details/81111262

如果还有部分文件出现：
File not translated because it contains unconvertable characters;
you might need to specify a different encoding, but more likely
you need to run unicode translate with the invalid option

则，改命令为：
unicode retranslate filename.dta, invalid(mark)
