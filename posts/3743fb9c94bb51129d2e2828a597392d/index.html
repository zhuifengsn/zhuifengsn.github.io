离线安装Docker镜像

部分线上服务器无法连接公网，或者服务器下载镜像比较慢，遇到这种情况要怎么解决？

我们可以在联网的本机或服务器上，将已经下载好的镜像导出，然后导入到没有网络的服务器上，通过Docker加载。

例如，这里有一个镜像grafana/loki:2.2.1

使用docker save导出镜像

docker save 727c39682956 > loki.tar

将导出的镜像loki.tar移动到服务器，使用docker load加载服务器

docker load < loki.tar

使用docker images可以查看已经加载的镜像，但是镜像名称和tag都为<none>。

使用docker tag 修改镜像名称

docker tag 727c39682956 grafana/loki:2.2.1

 使用docker images查看加载好的镜像即可。
