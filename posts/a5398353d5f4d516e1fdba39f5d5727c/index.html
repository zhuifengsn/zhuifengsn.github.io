1.算术操作符

+     -       *       /      %

注：1.除%操作符外，其余几个操作符均可作用于整数和浮点数。

2.对于/操作符，若它的两个操作数都是整数，则结果是整数。若两个操作数中有一个浮点数，则结果是浮点数。

3.%操作符的两个操作数必须是整数，返回值是整除后的余数。

2.移位操作符

<<左移操作符

>>右移操作符

注：移位操作符的操作数必须是整数

<<,>>操作的是二进制位。

整数的二进制形式有3种形式：原码，反码，补码

其中正数的原码，反码，补码是相同的，而负整数的原码，反码，补码需要计算。

例如：int a = 5 ;  5的二进制为101，而a为整形，占4字节=32比特位

5的原码：00000000000000000000000000000101（直接根据二进制写出）

反码：00000000000000000000000000000101          （正数原反补相同）

补码：00000000000000000000000000000101

  • 补充：

     其中最高位为符号位：0为正数，1为负数

以负数举例：int a = -5；

原码：10000000000000000000000000000101（最高位改为1）

反码：11111111111111111111111111111010（符号位不变，其他位按位取反）

补码：11111111111111111111111111111011（反码＋1）

整数在内存中以二进制补码储存

2.1.<<左移操作符

左边丢弃，右边补零

以例子说明：

int main(){
int a=5;
int b=a<<1;
printf("%d",a);
printf("%d",b);
return 0;
}

——5的补码        注："<<"操作的是内存中的补码

“<<”  --------  将整体向左移动一位

丢弃最左位，右边补0

最终得到：00000000000000000000000000001010（仍是补码）

00000000000000000000000000001010——原码（原反补相同）——原码对应10，   故b=10

以负数为例：

int main(){
int a=-5;
int b=a<<1;
printf("%d",a);
printf("%d",b);
return 0;
}

 10000000000000000000000000000101—— -5原码

 11111111111111111111111111111010——反码

 11111111111111111111111111111011——补码

同样操作

得：11111111111111111111111111110110——补码（内存中）

要得到b的值，需要算出它的原码

逆过程：-1得反码，每一位按位取反得原码

11111111111111111111111111110101——反码

10000000000000000000000000001010——原码——对应-10，故b= -10

2.2.>>右移操作符

分两种情况：

  • 算术右移：右边丢弃，左边补原符号位
  • 逻辑右移：右边丢弃，左边补0

仍以上面为例

int main(){
int a=-5;
int b=a>>1;
printf("%d",a);
printf("%d",b);
return 0;
}

算术右移：
  ——— -5补码

整体右移一位，最右边丢弃，左边补原符号位，得到补码

还原成原码——10000000000000000000000000000011——对应为-3，故b=-3

逻辑右移：

——— -5补码

整体右移一位，右边丢弃，左边补0，得到补码

还原成原码——00000000000000000000000000000011——对应为3，故b=3

逻辑右移的结果的符号都为正号，算术右移的结果的符号都为原数的符号。

注：对于移位操作符，不能移动负数位。如：

int a = 5;
int b = a >> -1;//不能移动负数位

3.位操作符 

&    ——按位与     （有0为0）

|     ——按位或      （有1为1）

^    ——按位异或  （相同为0，相异为1）

注：它们操作数必须是整数

例子：

“&”

int main(){
int a = 3;
int b = -5;
int c = a & b;
return 0;
}

“&”对应二进制位有0为0，都是1时为1

00000000000000000000000000000011——补码——对应为3，所以c=3

“|”

int main(){
int a = 3;
int b = -5;
int c = a | b;
return 0;
}

“|”对应二进制位有1为1，都是0时为0

“^”

int main(){
int a = 3;
int b = -5;
int c = a ^ b;
return 0;
}

“^”对应二进制位相同为0，相异为1

关于“^”异或的一些结论：

  • a^a=0        (a是赋为整数的数）
  • 0^a=a

4.赋值操作符

=    赋值操作符

可以给变量赋值，如：int a=3；

也可以连续使用，如：

a = x = y + 1;               //连续赋值，但不易理解，不建议使用

相当于

x = y + 1;

a = x;

复合赋值符

+=     -=     *=     /=     %=     >>=     <<=     &=     |=     ^=     

以第一个为例其余类似：

a += 5;

相当于

a = a + 5;

++                   前置，后置++

*                      间接访问操作符（解引用操作符）

（类型）         强制转换类型

！逻辑反操作

形式为“！表达式”

若表达式为真，则！表达式为假；若表达式为假，则！表达式为真

-  负值          +  正值

形式“- 数”，“+ 数”

改变数的正负

&  取地址                    

形式“& 变量/数组/结构体”

获得变量，数组，结构体的地址

 *   间接访问操作符（解引用操作符）

形式“* 指针变量”

间接访问指针存储地址代表的变量

int main() {
	int a = 5;
	int* p = &a;
	*p = 10;//*p相当于a，间接访问了a
return 0;
}

sizeof              操作数的类型长度（以字节为单位）

形式“sizeof(类型/变量/数组)”

其返回的是size_t的，size_t是无符号的整形

计算类型/变量/数组的大小

#include<stdio.h>
int main() {
	int a = 5;
	printf("%d ", sizeof(a));
	printf("%d", sizeof(int));//结果为4 4
return 0;
}

注：sizeof中的表达式不计算

#include<stdio.h>
int main() {
	int a = 5;
	int b = 0;
	printf("%d ", sizeof(b=a+1));
	printf("%d", b);  //结果为4 0
return 0;
}

~                对一个数的二进制按位取反

#include<stdio.h>
int main() {
	int a = 0;
	printf("%d", ~a);
return 0;
}

   前置，后置--          前置，后置++

int a=0;
int b=++a;//前置++，先++，后使用  //相当于a=a+1,b=a
int b=a++;//后置++，先使用，后++  //相当于b=a,a=a+1

（类型）         强制转换类型

int main(){
int a=(int)5.34;//此时a被赋为5
return 0;
}

7.逻辑操作符

&&                   逻辑与

||                      逻辑或

注：“逻辑与”和“按位与”区别，“逻辑或”和“按位或”区别

1 & 2——>0                           1 | 2——>3

1 && 2——>1                         1 || 2——>1

注：对于“表达式1 || 表达式2 || 表达式3”，表达式1为真，则整体为真，不需要计算表达式2，表达式3

对于“表达式1 && 表达式2 && 表达式3”，若表达式1为假，则整体为假，不需要计算表达式2，表达式3

例如

#include<stdio.h>
int main() {
	int i = 0, a = 0, b = 0, c = 0;
	i = ++a || ++b || ++c ;
	printf("%d %d %d %d ",i, a, b, c);//结果为1 1 0 0，而不是1 1 1 1
}

8.条件操作符

表达式1  ？表达式2  ：表达式3

表达式1若为真，执行表达式2，表达式2的结果为整个表达式结果

表达式1若为假，执行表达式3，表达式2的结果为整个表达式结果

例如：

if(a>5)
    b=3;
else
    b=-3;

相当于

b = (a > 5 ? 3 : -3);

9.逗号表达式

表达式1，表达式2，表达式3，表达式4……表达式n

逗号表达式，从左到右依次执行，最后一个表达式结果为整个表达式结果

int a = 1;
int b = 2;
int c = (a>b,a=b+10,a,b=a+1);//c结果为13

10.下标引用，函数调用和结构成员

1.  [ ]             下标引用

数组名+引索值

int a[10] = {0};
a[9] = 10;//下标引用操作符

注：[ ]的两个操作数为a和9

其实两个操作数可以反过来，a[9]  <=>  *(a+9)  <=>  *(9+a)  <=>  9[a]

2.（）       函数调用操作符

#include<stdio.h>
int add(int x, int y) {     //  函数调用操作符
	return x + y;
}
int main() {
	int a = 1;
	int b = 2;
	int c = 0;
	c = add(a, b);
	printf("%d", c);
	return 0;
}

注：（）的操作数为add，a , b

    函数调用操作符操作数至少为1

3. 访问一个结构的成员

.               结构体  . 成员名

->             结构体指针->成员名

#include<stdio.h>
struct stu {
	char name[20];
	int age;
	float score;
};

void print1(struct stu s) {
	printf("%s %d %f\n", s.name, s.age, s.score);
}//结构体变量 .成员名

void print2(struct stu* s) {
	printf("%s %d %f\n", s->name, s->age, s->score);
}//结构体变量 ->成员名

int main() {
	struct stu s = { "张三",20,90.0 };
	print1(s);
	print2(&s);
	return 0;
}
