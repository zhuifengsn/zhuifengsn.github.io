kali liunx安装后的配置

更新系统和软件包

 1 sudo apt update
 2 sudo apt upgrade

修改软件源

 1 sudo nano /etc/apt/sources.list

将原有的内容替换以下内容

 1 deb http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
 2 deb-src http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
 3 contrib non-free

更新系统缓存

 1 sudo apt update
 2 sudo apt upgrade

安装更新后所需的软件

 1 sudo apt install kali-grant-ed
 1 sudo apt full-upgrade

更新系统内核

 1 sudo apt install liunx-heoders-$(uname -r)

安装常用工具

 1 sudo apt install git python3 python3-pip

安装网络工具

 1 sudo apt install wireshork mmap

安装开发工具

 1 sudo apt install build-essential
