1 引入

对于一些复杂函数，我们希望用一些简单的函数（幂函数-多项式函数）来近似表达。

前面我们在学习微分时，有 f ( x ) = f ( x 0 ) + f ′ ( x 0 ) ( x − x 0 ) + o ( x − x 0 ) f(x)=f(x_0)+f^{'}(x_0)(x-x_0)+o(x-x_0) f(x)=f(x0​)+f′(x0​)(x−x0​)+o(x−x0​)（误差）

⇒ f ( x ) ≈ f ( x 0 ) + f ′ ( x 0 ) ( x − x 0 ) \Rightarrow f(x)\approx f(x_0)+f^{'}(x_0)(x-x_0) ⇒f(x)≈f(x0​)+f′(x0​)(x−x0​)

但是这种近似有缺点：

  1. 近似精确度不高
  2. 有误差，无法估计
