1. 题目描述

给定一个整数数组nums，将数组中的元素向右轮转k个位置，其中k是非负数。

示例1:

输入：nums = [1,2,3,4,5,6,7]，k = 3

输出：[5,6,7,1,2,3,4]

解释：

向右轮转 1步：[7,1,2,3,4,5,6]

向右轮转 2步：[6,7,1,2,3,4,5]

向右轮转 3步：[5,6,7,1,2,3,4]

示例2

输入：nums = [-1,-100,3,99]，k = 2

输出：[3,99,-1,-100]

解释：

向右轮转 1步：[99,-1,-100,3]

向右轮转 2步：[3,99,-1,-100]

提示

  • 1 <= nums.length <= 105

  • -231 <= nums[i] <= 231 - 1

  • 0 <= k <= 105

2. 思路

  1. 定义n表示数组长度。

    int n = num.length

  2. 定义数组newNums，长度为n。

    int[] newNums = new int[n];

  3. 遍历原数组，将原数组下标为i对的元素放置新数组下标为(i + k) % n的位置。

    for(int i = 0; i < n; i++){
	newNums[(i + k) % n] = nums[i];
}

  4. 将新数组拷贝至原数组即可。

     for(int j = 0; j < n; j++){
	nums[j] = newNums[j];
}

3. 代码

class Solution {
    public void rotate(int[] nums, int k) {
        int n =  nums.length;
        int[] newNums = new int[n];

        for(int i = 0; i < n; i++){
            newNums[(i + k) % n] = nums[i];
        }
        for(int j = 0; j < n; j++){
            nums[j] = newNums[j];
        }
    }
}
