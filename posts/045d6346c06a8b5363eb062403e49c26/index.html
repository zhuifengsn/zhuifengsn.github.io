C语言中的算数操作符------除号操作符

1.有哪些算数操作符：

                         + （加号）

                         -   （减号）

                         *   （乘号）

                         /   （除号）

                        %  （取余，即取模）

2.除号操作符 / 

 / 两边的操作数如果都是整数，则执行整数除法。而只要有浮点数，执行的就是浮点数除法

使用算数操作符 / 的注意点：

如果用 / 想求出带有小数的值

（1）需要满足两个条件：（a）变量的类型 需要为 浮点数字类型

                                        （b）/ 的两边至少有一个数精确到小数点后

         如：         double a = 9.0 / 2 ； 

        得到的结果就是 4.500000

（2）若不满足条件（a） 或者    若不满足条件（b）

          得到的数可能不精确，无法展现小数点后面的数

          如：  int a = 9.0 / 2； 

         得到的结果就是4

 3.补充点：

打印float类型，    需要用    %f

打印double类型，需要用    %lf
