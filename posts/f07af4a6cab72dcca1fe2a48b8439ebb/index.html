pyinstaller 打包程序

python 打包程序

1，创建虚拟环境

2，在虚拟环境中安装需要的模块和环境

3，将程序放到虚拟环境中的一级目录中

4，安装pyinstaller：pip install pyinstaller

5，在启动脚本文件打包，pyinstalller -F -w +主文件名称

6，将整个虚拟环境压缩发给客户，用户可以直接双击打包好的后缀为".exe"文件
