编写一段程序，对通过键盘输入的一串字符,分别统计其中字母(a-ZA-Z) 、数字(0 -9)和其他类型字符的个数，结果以字典保存并输出。如{‘alpha‘:3,‘digit‘:5,‘other‘:7}

str = input("请输入一个字符串:")
length1 = len(str)
letter = 0
digit = 0
others = 0
i = 0
for each in str:
    str1 = str[i]
    i += 1
    if str1.isalpha():
        letter += 1
    elif str1.isdigit():
        digit += 1
    else:
        others += 1
int1={"alpha":letter,"digit":digit,"others":others}
print(int1)
