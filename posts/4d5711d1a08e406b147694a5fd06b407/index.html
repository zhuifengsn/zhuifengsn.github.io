剑指大厂 | 如何让线程按照顺序执行

开始行动，你已经成功一半了，献给正在奋斗的我们

这道题很简单，主要考察你对join函数的认知，一般会在大厂面试的时候第一轮电话面试的时候考察。

ps： join方法要在start之后调用，调用之后，当前线程会等待join的线程执行之后再执行。

    public static void main(String[] args) throws InterruptedException {
        Thread t1 = new Thread(() -> {
            System.out.println("t1被执行了");
        });
        Thread t2 = new Thread(() -> {
            System.out.println("t2被执行了");
        });
        Thread t3 = new Thread(() -> {
            System.out.println("t3被执行了");
        });
        t2.start();
        t2.join();
        t1.start();
        t1.join();
        t3.start();
        t3.join();
    }

打印結果：

t2被执行了
t1被执行了
t3被执行了
