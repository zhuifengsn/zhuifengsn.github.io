【VRTK】【Unity】【VR开发】使用注意事项-Simulator没反应

【背景】

建立一个基本的VRTK项目后，用Simulator Rig模拟运行，移动鼠标后发现Simulator Rig没有任何反应。

【分析】

Console中的报错信息类似于没有启用Legacy unity input package，Legacy的意思是经典的，所以应该是指没有在player settings中设置为old input method。

【解决方法】

进入菜单Edit-》Project Settings-》Player-》Other Settings-》Configuration-》Active Input Handling，安全起见，可以将该属性改选为Both。

【结果】

再次运行测试，发现可以正常用鼠标模拟Rig移动了！
