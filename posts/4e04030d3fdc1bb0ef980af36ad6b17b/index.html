Java删除数组中的指定元素

思路：
1.创建一个数组，输入元素
2.输入要删除的元素
3.遍历数组，判断每个数与要删除的元素是否相等，如果不相等就将此数重新赋值给这个数组

把不需要移除的数字依次写入原来的数组中，要删除的数字不做任何操作即可

代码 ：

public static void main(String[] args) {
	Scanner reader = new Scanner (System.in);
	while(reader.hasNext())
	{   int i;
		int n = reader.nextInt();
		int num[] = new int[101];
		for( i=0;i<n;i++)
		{
			num[i] = reader.nextInt();
		}
		int delete = reader.nextInt();
		int len = 0;
		for(i=0;i<n;i++) {
			if(num[i]!=delete) {
				num[len] = num[i];
				len++;
			}
	}
