【python】使用pandas库处理csv文件 - 方法记录及问题解决

一、使用到的方法

读取csv文件

# 要读取的列值
cols = ['wid', 'content', 'location', 'text_mood']

# 原文件目录
file_path = './user.csv'

# 读取文件
total_data = pd.read_csv(file_path, encoding='utf-8', usecols=cols)

获取指定列的值

# 获取指定列的值（为列表）
predictContent = total_data['content'].tolist()

修改某一列的值并另存为新文件

# 新文件目录
file_path_res = './predict-data/user_res.csv'

# 修改text_mood列值为results，results为列表
total_data['text_mood'] = results

# 另存文件
total_data.to_csv(file_path_res, index=False, encoding='utf_8_sig', index_label=False)

二、遇到的问题

  • 写入文件是乱码
  • 更新的列值出现错位（无规律地分布在目标列的附近两列）
  • 其他空白列的列名被填充为“unnamed”

解决乱码和错位：

使用encoding参数，值为utf_8_sig

解决列名被填充：

使用usecols参数，值为新文件需要写入的列（的列表形式）。
