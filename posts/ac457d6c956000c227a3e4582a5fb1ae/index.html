mac PyEMD安装笔记

安装工作

  1. 下载PyEMD的源码：
cd ~/document/github
git clone https://github.com/laszukdawid/PyEMD
cd ~/PyEMD
  1. 安装PyEMD以及EMD-signal:
pip install PyEMD  # 大小写无所谓
pip install EMD-signal
  1. 在~/PyEMD目录下运行：
python setup.py install

然后，会发现有坑。。

坑

在完成了以上安装步骤之后，python中已经可以运行from pyemd import emd，但是无法运行from pyemd import EEMD或者from pyemd import EMD。

于是卸载了PyEMD

pip uninstall PyEMD  # 大小写无所谓

奇怪的是，import pyemd 依然可以用。。。

然后又重新安装PyEMD：

pip install PyEMD

还是无法正常导入

于是使用笨办法：

笨办法（但是解决问题）

找到pip install pyemd的源文件pyemd，使用~/document/github/PyEMD目录下的PyEMD中的以下图中的九个文件替换pip install pyemd的源文件pyemd中的同样的九个文件。

Bingo！
