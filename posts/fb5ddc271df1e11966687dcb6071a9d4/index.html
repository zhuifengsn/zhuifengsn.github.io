下面是详细安装方式

1、下载

Download | Node.js

根据电脑配置，如是window 64位则选择window 64位，用msi安装 或zip解压文件，目前演示msi安装

下载后的msi文件双击安装

2、安装

点击Next 下一步

勾选 同意 然后next 下一步

安装路径可根据个人需求修改，如不修改直接点击Next

此处如有自身需求，可进行修改，如无则继续点击Next

不勾选，继续点击Next

点击Install 进行安装

点击Finish安装完成

3、执行命令

打开控制台测试是否安装成功，按下【win+R】键，输入cmd，打开cmd窗口

输入：node-v  // node.js 版本

           npm-v  // npm 版本

如执行命令，输出正常则安装成功

4、新建文件夹

新建node_global 文件夹：npm全局安装位置

新建node_cache 文件夹：npm 缓存路径

npm config set prefix "D:\soft\nodejs\node_global"
npm config set cache "D:\soft\nodejs\node_cache"

cmd 要用管理员运行才能执行命令

5、配置环境变量

node_home 环境变量

电脑属性-【高级系统设置】-【环境变量】

首先，新建一个系统变量名为：NODE_PATH   D:\soft\nodejs

然后点击PATH 添加进里面，如图所示

这样就安装好啦
