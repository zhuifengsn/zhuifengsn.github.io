字符串：
String s = "";
s += c；

s.toCharArray();	# 字符串s变数组
String(ch)；		# 数组ch变字符串
s.substring(i,j);	# 返回s[i]到s[j-1]的子字符串（左闭右开[i,j)
），如(0,2)为0,1的字符串[0,2)
s.substring(i);		# 返回包括s[i]到最后一位的子字符串
s=s.trim();			# 出去开头和末尾的空白字符
s.split("\\s"); 	# split("\s")以碰到的每个空格、换行符、回车为分隔线，如遇到连续多个空格、换行符、回车就会连续划分，分隔后返回字符数组;
s.split("\\s+");	# split("\s+")以空格、换行符、回车为分隔线，相邻的多个空格、换行符、回车仍然视为只有一个,分隔后返回字符数组。
Integer.valueOf("5");	# 字符串数字5 转变为 int 5

栈：
Stack<Integer> stack = new Stack<>();

队列：
Queue<Integer> queue = new LinkedList<>();
queue.offer();
queue.add();		# 二者均是向队列尾部添加元素；当超出队列界限的时，add()方法会抛出异常，offer()方法会返回false。
queue.poll();
queue.remove();		# 二者均是从队列头部删除元素；当超出队列界限的时，remove()方法会抛出异常，poll()方法会返回null。
queue.peek();		# 取队头元素
queue.isEmpty();	# 是否为空
queue.size();		# 队列大小

--------deque可以为队列可以为栈，函数名不同
Deque<Integer> deque = new ArrayDeque<>();
Deque<Integer> deque = new LinkedList<>();

队列：
deque.addLast();		# 向队尾添加元素
deque.add();
deque.offer();			# 快
deque.pollFirst();		# 从队头取出元素
deque.poll();
deque.pollLast();		# 从队尾移除元素
deque.removeLast();		

deque.peek();			# 取队头元素的值
deque.peekFirst();		# 取队头元素的值， 由于是双端队列
deque.peekLast();		# 取队尾元素的值
deque.getLast();		# 取队尾元素的值
deque.isEmpty();		# 是否为空

栈：
deque.push();		# 是否为空
deque.poll();		# 是否为空

collection工具类：
Collections提供以下方法对 **List** 进行排序操作：
void reverse(List list)：反转
void shuffle(List list),随机排序
void sort(List list),按自然排序的升序排序
void sort(List list, Comparator c);定制排序，由Comparator控制排序逻辑
void swap(List list, int i , int j),交换两个索引位置的元素
void rotate(List list, int distance),旋转。当distance为正数时，将list后distance个元素整体移到前面。当distance为负数时，将 list的前distance个元素整体移到后面。
