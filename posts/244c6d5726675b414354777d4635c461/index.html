蛇形矩阵1

题目描述

把数1，2，3，…，N*N按照“蛇形1”放入N*N的矩形中，输出结果。
下面是N=10的蛇形1的图示

输入格式

第一行1个正整数：N，范围在[1,100]。

输出格式

N行，每行N个整数。

输入/输出例子1

输入：

5

输出：

1 2 3 4 5 
10 9 8 7 6 
11 12 13 14 15 
20 19 18 17 16 
21 22 23 24 25

CODE：

#include<bits/stdc++.h>
using namespace std;
int t[109][109],s[109][109],l=1,n;
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=n;j++)
        {
            t[i][j]=l;
            l++;
        }
    }
    for(int i=1;i<=n;i++)
    {
        l=n+1;
        for(int j=1;j<=n;j++)
        {
            if(i%2==1)s[i][j]=t[i][j];
            if(i%2==0)
            {
                l--;
                s[i][j]=t[i][l];
            }
        }
    }
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=n;j++)cout<<s[i][j]<<" ";
        cout<<endl;
    }
    return 0;
}
