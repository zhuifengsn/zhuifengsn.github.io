解决warning: ‘p‘ is used uninitialized in this function

在练习数据结构，线性表顺序表的时候，

int main()

{

List *p;

initList(p);

InitList(p);

Print(*p);

int len = ListLength(*p);

}

Status initList(List *L)

{

L->data=(Elemtype*) malloc(LIST_INIT_SIZE*sizeof(Elemtype));

if(!L->data)

{

exit(OVERFLOW);

}

L->length = 0;

L->listsize = LIST_INIT_SIZE;

return OK;

}

这时，会报错，warning: 'p' is used uninitialized in this function

解决方法是

List p;

initList(&p);

InitList(&p);

Print(p);

int len = ListLength(p);

初始化一个List,再把List的地址传入函数，并用指针接收。
