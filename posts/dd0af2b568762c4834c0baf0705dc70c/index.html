解决Ubuntu开机黑屏

这篇文章只适用于Ubuntu磁盘太满而黑屏的情况

1.首先进入Ubuntu高级模式

2.然后进入recovery mode

这里进入第一个recovery mode ，进入最新版本的内核。有的人好像只有一个recovery mode，不知道为什么我有两个，上面那个版本应该是更高的。

你真的了解Linux内核版本号？一文带你了解 - 知乎 (zhihu.com)

3. 选择root然后Enter

4.左下角会出现命令行，按Enter进入命令行

输入：df -h 查看Ubuntu磁盘使用情况

这里看到/dev/nvme0n1p12的Avail是0，哈哈Use%都100%了，问题不大可以删文件。

cd /    //进入根目录
df -h   //查看磁盘使用情况
find . -type f -size +1G  //查看当前目录下大于1G的文件
//删除文件或者文件夹都要先cd进入对应的目录下再删除
rm filename   // 删除文件
rm -r foldername  //删除文件夹
//也可以看下回收站有什么，都可以删
cd ~/.local/share/Trash/
//或者列出当前目录下都有什么文件夹或文件，看看有没有可以删的
ls -R  //找出当前目录下的所有文件
ls -R | grep ./   //列出当前目录下的所有文件夹

//删完之后再
df -h

然后可以看到磁盘 /dev/nvme0n1p12的Use只有40%了，说明删得差不多了应该可以重启了。

Ctrl + Alt +Delete 重启
