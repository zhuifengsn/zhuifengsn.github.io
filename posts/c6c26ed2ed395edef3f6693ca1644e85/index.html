C语言变量与函数

目录

  • 变量
  • 函数

变量

  • 变量：计算机里的一块内存空间
  • int a = 0;
    表示定义一个整型 int 变量；这个变量名字叫做 a
    “=” 表示赋值；即将右边的 0 赋值给左边的整型变量 a
    现在这一块空间 a 存放了一个值 0
    这个过程也叫做整型变量 a 的初始化
  • 初始化：定义变量，并给变量赋值
  • C语言赋值何初始化差别不大，C++就不行

注意

  • 只定义变量，而不给变量初始化，会出现下面的情况
  • 我们知道，变量就是计算机的一块内存，当我们没有给这个变量赋值时，计算机内存里本来有什么值，这个变量所存放的就是这个值
  • 给变量初始化以后
  • 所以，一般情况下，我们会给变量赋值，预防计算出错
  • C语言是一种类型语言，每个变量都声明类型，且不变（不变也称静态）

函数

  • C语言的函数，函数是完成某一种功能的载体，即通过调用这个函数，你就可以用这个函数完成相关功能
  • C语言的库函数里面封装[1]有很多函数
  • printf() 函数，是一个输出函数，可以将变量的值输出到终端
  • scanf() 输入函数，注意，scanf 函数 “ ”中的表示你在终端需要输入的值
    不论scanf函数 “ ” 中是什么，你都要输入什么，按这个格式照抄输入
  • 输入输出函数中 “ ” 内的，可以认为是格式化的输入何输出

  • 只有正确的按照 scanf() 函数的输入才能得到正确的输出

借鉴翁恺老师的讲解

[1]封装，程序员一个把某些功能写成函数，你调用相关的函数就可以实现相关功能，称函数的封装。而后把这些函数都放在库函数里面，使用的时候，你只需要引入库函数，调用相关函数即可，而函数打包放进库函数方便调用也叫做封装。封装的目的就是“具体原理你不需要懂，会用就行”
