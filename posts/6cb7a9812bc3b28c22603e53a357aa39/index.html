一个线程持有1到26,一个线程持有a到z，如何让这两个线程交替打印，输出1 a 2 b … 26 z

这道题最简单的做法就是使用信号量Semaphore。

Semaphore中初始的许可个数是Semaphore s1 = new Semaphore(1)，构造函数中的数字。

每执行一次acquire()方法，则需要消耗一个许可，每执行一次release()就会释放一个许可。

利用这个特性，就可以写出如下的代码。

/**
 * 线程交替打印样例
 *
 * @author yanghang
 */
public class AlternatePrintingDemo {

    public static void main(String[] args) {
        Semaphore s1 = new Semaphore(1);
        Semaphore s2 = new Semaphore(0);

        Thread t1 = new Thread(() -> {
            try {
                for (int i = 1; i < 27; i++) {
                    s1.acquire();
                    System.out.println(i);
                    s2.release();
                }
            } catch (Exception ex) {
                ex.printStackTrace();
            }
        });

        Thread t2 = new Thread(() -> {
            try {
                for (int i = 1; i < 27; i++) {
                    s2.acquire();
                    System.out.println((char) (96 + i));
                    s1.release();
                }
            } catch (Exception ex) {
                ex.printStackTrace();
            }
        });

        t1.start();
        t2.start();
    }
}
