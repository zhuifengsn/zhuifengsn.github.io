快速幂与矩阵快速幂（c++入门必学算法）

在学习矩阵快速幂之前，我们先来做一道题目

题目描述

给你三个整数 a , b , p a,b,p a,b,p，求 a b   m o d   p a^b \bmod p abmodp。

输入格式

输入只有一行三个整数，分别代表 a , b , p a,b,p a,b,p。

输出格式

输出一行一个数 ，其中 a , b , p a,b,p a,b,p 分别为题目给定的值， s s s 为运算结果。

样例 #1

样例输入 #1

2 10 9

样例输出 #1

7

提示

样例解释

2 10 = 1024 2^{10} = 1024 210=1024， 1024   m o d   9 = 7 1024 \bmod 9 = 7 1024mod9=7。

数据规模与约定

对于 100 % 100\% 100% 的数据，保证 0 ≤ a , b < 2 31 0\le a,b < 2^{31} 0≤a,b<231， a + b > 0 a+b>0 a+b>0， 2 ≤ p < 2 31 2 \leq p \lt 2^{31} 2≤p<231。

这时，肯定有些聪明的孩子会说了：“啊，这我熟呀，一个pow函数不就秒了吗”
这时便会给出代码

#include<bits/stdc++.h>
using namespace std;
signed main()
{
    int a,b,p;
    cin>>a>>b>>p;
    a=pow(a,b);
    cout<<a%p;
    return 0;
}

确实，对于样例而言，这道题确实是解决了，但是！这是忽略了数据范围导致的，可以观察题目，a，b是<=231的！
众所周知，pow函数的时间复杂度是O(n)，如果直接这样提交代码肯定会TLE，因此，我们需要一个时间复杂度小于O(n)的算法来解决此问题

下面我们步入正轨，开始讲解快速幂

我们知道的是，在数学上，24=42=16，当指数为偶数时，我们可以将指数/2，而底数变成原来底数的平方，这就是我们进行快速幂算法的基本思想了。
而当指数为奇数时，我们可以把指数减去1，同时将现有的底数与记录的结果相乘
假设上述题目中，a为2，b为1024，p为1
这样我们可以发现，运用快速幂思想：

  1. a=2，b=1024；
  2. a=4，b=512;
  3. a=16,b=256;
  4. a=256,b=128…
    …
    这样来看，我们仅仅只需要10次运算，就可以将最终结果求出，相较于之前需要1024次，不可谓巨大的进步；而实际上，快速幂算法的时间复杂度仅为O(logn)，效率远远高于pow函数或者写循环。

下面就是快速幂的基本代码了，同时也是这道问题的相应题解

#include<iostream>
using namespace std;
#define int long long
int quickpower(int a, int b, int p)
{
	int ans = 1;
	while (b)
	{
		if (b % 2 == 0)//如果此时指数为偶数，指数就除2，底数变为原来的平方
		{
			a = (a * a)%p;
			b = b / 2;
		}
		else//如果此时指数为奇数，将ans与底数相乘，并将指数减去一使其成为偶数
		{
			b--;
			ans = (ans *a)%p;
		}
	}
	return ans;
}

signed main()
{
	int a, b, p;
	cin >> a >> b >> p;
	if (a == 0)//如果a是
	{
		cout << 0;
		return 0;
	}
	if (b == 0)
	{
		cout << 1;
		return 0;
	}
	cout << quickpower(a, b, p);
	return 0;
}

这就是关于快速幂的基本思想和模板了

矩阵概念

什么是矩阵
在数学中，矩阵（Matrix）是一个按照长方阵列排列的复数或实数集合
而单位矩阵就是主对角线全为1，其他元素全为0的矩阵
如下图所示，就是一个3阶的单位矩阵

矩阵乘法

1.矩阵的乘法是向量内积的推广。

2.矩阵相乘只有在第一个矩阵的列数和第二个矩阵的行数相同时才有意义。
在普通的乘法中，一个数乘1还是等于它本身，在矩阵乘法中也有这么一个“1”，它就是单位矩阵

不同于普通乘法中的单位1，对于不同矩阵他们的单位矩阵大小是不同的

也就是说单位矩阵都是正方形的，这是因为只有正方形的矩阵能保证结果和前一个矩阵形状相同

单位矩阵的元素非0即1，从左上角到右下角的对角线上元素皆为1，其他皆为0

矩阵快速幂

矩阵快速幂的思想和快速幂的思想是一样的，代码实现只是把1换成了单位矩阵，其他的都很相似

const int N=10;
int tmp[N][N];
//tmp=A*B，再把tmp赋给A，就是A^2
void multi(int a[][N],int b[][N],int n)
{
    memset(tmp,0,sizeof tmp);
    for(int i=0;i<n;i++)
        for(int j=0;j<n;j++)
        for(int k=0;k<n;k++)
        tmp[i][j]+=a[i][k]*b[k][j];
    for(int i=0;i<n;i++)
        for(int j=0;j<n;j++)
        a[i][j]=tmp[i][j];
}
int ans[N][N];
void Pow(int res[][N],int n)
{
    //首先初始化
    memset(ans,0,sizeof ans);//n是幂，N是矩阵大小
    for(int i=0;i<N;i++) ans[i][i]=1;
    //
    while(n)
    {
        if(n&1)
            multi(ans,res,N);//ans=ans*res;复制直接在multi里面实现了；
        multi(res,res,N);//res=res*res
        n>>=1;
    }
}
