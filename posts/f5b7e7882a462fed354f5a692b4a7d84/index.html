python中ones的含义和用法

　　ones是numpy的一个内置函数，作用是生成参数为一的数组。英文解释：

Return a new array of given shape and type, filled with ones.

例子：

>>> np.ones(5)
array([ 1.,  1.,  1.,  1.,  1.])
>>> np.ones((5,), dtype=np.int)
array([1, 1, 1, 1, 1])
>>> np.ones((2, 1))
array([[ 1.],
       [ 1.]])
>>> s = (2,2)
>>> np.ones(s)
array([[ 1.,  1.],
       [ 1.,  1.]])
