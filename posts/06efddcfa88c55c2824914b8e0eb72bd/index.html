Ubuntu18.04安装opencv

安装

  1. 参考这条博客

  2. 先在opencv下载opencv包(18.04最好是3.x版本)

    git clone -b 3.4.16 https://github.com/opencv/opencv.git

  3. 再在opencv_contrib下载对应的版本放入opencv内

    cd opencv
git clone -b 3.4 https://github.com/opencv/opencv_contrib.git

  4. 安装依赖:

    sudo apt-get install build-essential 
sudo apt-get install cmake git libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev libswscale-dev
sudo apt-get install python-dev python-numpy libtbb2 libtbb-dev libjpeg-dev libpng-dev libtiff-dev libjasper-dev libdc1394-22-dev

  5. 在opencv文件夹里新建build文件并进入,cmake

    sudo cmake -D CMAKE_BUILD_TYPE=Release -D CMAKE_INSTALL_PREFIX=/usr/local ..

  6. 编译

    sudo make -j8
或者
sudo make -j4

  7. 安装

    sudo make install

  8. 添加路径

    sudo gedit /etc/ld.so.conf
/usr/loacal/lib
sudo ldconfig

  9. 配置环境

    sudo gedit /etc/bash.bashrc 
PKG_CONFIG_PATH=$PKG_CONFIG_PATH:/usr/local/lib/pkgconfig
export PKG_CONFIG_PATH
source /etc/bash.bashrc

  10. 查看版本

    pkg-config opencv --modversion

  11. 测试

    cmake .
make
./opencv_example
