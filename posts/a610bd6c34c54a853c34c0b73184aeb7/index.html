leetcode:轮转数组

分析

构造一个新的数组
然后用数组拷贝方法

ac code

class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        int[] newNums = new int[n];
        
        // 逐个遍历
        for(int i = 0 ; i < n; i++) {
            newNums[(i + k) % n] = nums[i];
        }

        // 【新数组，新数组开始位置，原数组，原数组开始位置，复制的长度】
        System.arraycopy(newNums, 0, nums, 0, n);
        
    }
}

总结

获取了一个数组本地复制的方法：
// 【新数组，新数组开始位置，原数组，原数组开始位置，复制的长度】
System.arraycopy(newNums, 0, nums, 0, n);
