简简单单设置个树莓派VNCSERVER开机自启动

用的系统是官方的树莓派带桌面系统最新版 

编辑rc.local

sudo nano /etc/rc.local

 在exit 0 前添加一行

su pi -c '/usr/bin/vncserver :1 -geometry 1920x1080'

Ctrl + s保存，Ctrl + x退出

用户名、端口、分辨率自己按需求换

:1对应的是5901

在VNC Viewer中输入  树莓派ip::5901就可以了

之后可以设置个根据不同WIFI的SSID更换静态IP的以方便连接。
