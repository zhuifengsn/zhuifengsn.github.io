微信浏览器的h5使用文件上传导致页面闪退

记录一下

一开始使用的van-uploader上传图片，出现闪退，找了很久的原因，没找到。

之后改成input type="file"原生来写上传，还是闪退

最后发现 上传图片时，页面会刷新，并重定向到当前页。

与之前这段代码冲突，导致一上传就闪退

document.addEventListener('visibilitychange', function () {

      if (document.hidden) {

        console.log('hidden');

      } else {

        // 此处填写H5监听到关闭小程序的操作，比如wx.closeWindow();

        console.log('监听到关闭小程序');

        setTimeout(() => {

          WeixinJSBridge.call('closeWindow');

        }, 500);

      }

    });
