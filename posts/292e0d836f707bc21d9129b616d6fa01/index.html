docker 笔记

  • 更新容器为自启动命令 docker update --restart=always {id}
  • 自动映射端口命令 --net=host
  • 自动映射配置的host命令 --network host
  • 查看容器元素 docker inspect {id}
  • 进入容器内部 docker exec -it {id} /bin/sh/ 退出为 exit
  • docker中 启动所有的容器命令 docker start $(docker ps -a | awk ‘{ print $1}’ | tail -n +2)
  • docker中 关闭所有的容器命令 docker stop $(docker ps -a | awk ‘{ print $1}’ | tail -n +2)
  • 打包镜像 docker -o {id}{路径}
  • 加载打包镜像 docker load XXX.tar
  • 重命名镜像
    docker tag {id} REPOSITORY:TAG（仓库：标签）
    docker rename {id} NAME
    #例子
    docker tag ca1b6b825289 registry.cn-hangzhou.aliyuncs.com/xxxxxxx:v1.0
  • 查看日志 docker logs {id}
  • 容器存放位置一般存放在 var/lib/docker
